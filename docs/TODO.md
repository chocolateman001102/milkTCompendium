# 项目待做事项

这个文件用于集中记录 MilkTCompendium 项目内的待做事项。之后新增、发现或暂缓的待办都放在这里，避免散落在聊天记录或代码注释里。

## 待设计

- 共饮页趋势图精确选择交互：当前先移除“粗点区域 -> 精确候选条”方案。后续需要重新设计一个更优雅的精确选择方式，既适合手指触控，又不让趋势图承担过重的点选负担。
- 已下架饮品复核与处理流程：需要明确已经下架的饮品如何复核、如何标记，以及是否在列表、搜索、档案页中隐藏或保留提示。
- 左上角 UI 项数文案：检查当前左上角显示“几项”的位置、含义和中文文案是否准确。
- 搜索结果展示页优化：梳理搜索结果页的信息层级、排序、空状态、长文本展示和可点击区域。
- 长名称相邻展示问题：两个长名字挨在一起时评估是否需要换位置、换行或调整布局，避免视觉拥挤。
- 错误品牌名清理：排查并修正错误品牌名，必要时建立品牌名校验或映射规则。
- 档案分热度表达：评估是否在档案分旁增加热度条、热度等级或类似可视化指标。

## 待验证

- 贴纸生成效果真机回归：当前先采用无橙色白边、轻灰影和原图细节优先的处理方案；后续用更多暗光、逆光、包装文字较小的样张确认清晰度、描边观感、处理耗时和 PNG 文件大小。
